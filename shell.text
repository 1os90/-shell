في نظام Linux

1. Bourne Shell (sh):

الشيل الأصلي في Unix. يدعم أوامر البرمجة الأساسية.



2. Bourne Again Shell (bash):

النسخة المحسنة من Bourne Shell، الأكثر شعبية في توزيعات Linux. يدعم ميزات مثل التكملة التلقائية والتحكم في الوظائف.



3. Korn Shell (ksh):

يجمع بين ميزات Bourne وC Shell. يقدم تحسينات في البرمجة وتفاعل المستخدم.



4. Z Shell (zsh):

شيل متطور مع ميزات تخصيص متقدمة وتكملة تلقائية ذكية. شائع بين المطورين.



5. C Shell (csh):

يعتمد على بنية لغة C، يدعم ميزات تفاعلية مثل التحكم في التاريخ.



6. TC Shell (tcsh):

نسخة محسنة من C Shell، مع ميزات إضافية مثل التكملة التلقائية.



7. Fish Shell (fish):

شيل موجه للمستخدمين الجدد، يوفر واجهة سهلة الاستخدام وميزات ذكية.




في نظام Windows

1. Command Prompt (cmd.exe):

الشيل التقليدي في Windows، يوفر واجهة سطر أوامر بسيطة.



2. PowerShell:

شيل متقدم تم تطويره لتوفير بيئة غنية بالبرمجة. يدعم Cmdlets، ويمكّن من إدارة النظام بفعالية.



3. Windows Subsystem for Linux (WSL):

يسمح بتشغيل توزيعات Linux داخل Windows. يدعم استخدام Bash وsh وzsh وغيرهم.



4. Git Bash:

يأتي مع Git for Windows، ويوفر بيئة Bash لتشغيل الأوامر على نظام Windows.



5. Cygwin:

بيئة شيل توفر أدوات Unix/Linux على Windows، مما يتيح للمستخدمين استخدام أوامر Linux.
